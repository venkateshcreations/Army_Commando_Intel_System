# Army Commando Intelligence System (ACIS)

A comprehensive military intelligence dashboard system for command operations, surveillance, and tactical planning.

## Features

- **Dashboard** - Real-time overview of mission status, alerts, and key metrics
- **Tactical Map** - Interactive map with unit positioning and terrain visualization
- **Surveillance** - Live surveillance feeds and sensor data monitoring
- **Intelligence Feed** - Real-time intelligence updates and threat detection
- **Mission Planner** - Strategic mission planning and assignment tools
- **Unit Management** - Track and manage deployed units and personnel
- **Analytics** - Data analytics and performance metrics visualization
- **Weather** - Tactical weather information and forecasts
- **Communications** - Secure communication system interface
- **Biometrics** - Personnel biometric monitoring and tracking
- **Battle Damage** - Damage assessment and casualty tracking
- **Alerts** - Priority alert system for critical notifications
- **Settings** - System configuration and preferences

## Tech Stack

- **Frontend**: React 18 + Vite
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Charts**: Recharts
- **Maps**: Leaflet + React-Leaflet
- **Data Visualization**: D3.js
- **Icons**: Lucide React

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure
